# HU-087

Tautan produk:

- [HU-087](https://fr.aliexpress.com/item/1005010207331462.html)

Papan ini termasuk salah satu kit berbiaya rendah dengan bentuk seperti jam tangan.

## Pemrograman

Sebelum merakit produk, lepaskan terlebih dahulu baterai yang direkatkan ke papan agar header pemrograman dapat diakses. Anda memerlukan dongle pemrograman `ESP32` yang bisa mengendalikan pin `EN` dan `IO0`.

Konektor USB pada perangkat ini tidak terhubung ke jalur USB bawaan `ESP32-S3`, sehingga hanya dapat dipakai untuk pengisian daya.
