# Waveshare ESP32-P4-NANO

`ESP32-P4-NANO` adalah papan pengembangan berukuran ringkas dan terintegrasi tinggi dari Waveshare yang memakai chip `ESP32-P4`.

Tautan produk:

- [ESP32-P4-NANO](https://www.waveshare.com/esp32-p4-nano.htm)

## Layar yang direkomendasikan

| Produk | Dependensi | Status |
|---|---|---|
| [10.1-DSI-TOUCH-A](https://www.waveshare.com/10.1-dsi-touch-a.htm) | [waveshare/esp_lcd_jd9365_10_1](https://github.com/waveshareteam/waveshare-ESP32-components/tree/master/display/lcd/esp_lcd_jd9365_10_1) | Sudah diuji |
| [101M-8001280-IPS-CT-K](https://www.waveshare.com/101m-8001280-ips-ct-k.htm) | [waveshare/esp_lcd_jd9365_10_1](https://github.com/waveshareteam/waveshare-ESP32-components/tree/master/display/lcd/esp_lcd_jd9365_10_1) | Sudah diuji |
