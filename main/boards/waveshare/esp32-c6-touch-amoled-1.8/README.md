# Waveshare ESP32-C6-Touch-AMOLED-1.8

`ESP32-C6-Touch-AMOLED-1.8` adalah papan pengembangan mikrokomputer dari Waveshare yang berukuran ringkas dan sangat terintegrasi. Papan ini memakai layar sentuh kapasitif `AMOLED` 1,8 inci, pengelola daya, `IMU` 6 sumbu, `RTC`, dan codec audio hemat daya.
