# Waveshare ESP32-S3-Touch-LCD-4B

`ESP32-S3-Touch-LCD-4B` adalah papan cerdas berbentuk kotak dinding 86 yang dirancang oleh Waveshare dengan modul `ESP32-S3` dan layar sentuh kapasitif `IPS` beresolusi `480x480`.

Tautan produk:

- [ESP32-S3-Touch-LCD-4B](https://www.waveshare.com/esp32-s3-touch-lcd-4b.htm)

## Konfigurasi

Atur papan lewat `menuconfig` pada jalur berikut:

```text
Xiaozhi Assistant -> Board Type -> Waveshare ESP32-S3-Touch-LCD-4B
```
