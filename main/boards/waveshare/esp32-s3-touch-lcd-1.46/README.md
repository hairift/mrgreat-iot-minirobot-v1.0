# Waveshare ESP32-S3 Touch LCD 1.46

Dokumentasi ini berlaku untuk varian `ESP32-S3-Touch-LCD-1.46` dan `ESP32-S3-Touch-LCD-1.46B`.

## Menentukan target

```bash
idf.py set-target esp32s3
```

## Membuka konfigurasi

```bash
idf.py menuconfig
```

Pilih tipe board `esp32-s3-touch-lcd-1.46` pada menu `Xiaozhi Assistant -> Board Type`.

## Kompilasi

```bash
idf.py build
```
